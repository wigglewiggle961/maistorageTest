## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| sm | `gap="small"` | Compact column spacing |
| md | `gap="medium"` | Default column/element spacing |
| lg | `gap="large"` | Wide column gaps |

Exceptions: Spacing in Streamlit is automatically handled by the framework grid. Native UI styling takes precedence over explicit pixel declarations.

## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | var(--font-size) | 400 | default |
| Label | var(--font-size-sm) | 500 | default |
| Heading | var(--font-size-lg) | 600 | 1.2 |

*Note: Streamlit system typography styling replaces arbitrary pixel mapping for this phase.*

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | System mode (Light/Dark) | App background, chat containers |
| Secondary (30%) | System mode alternate | Assistant message chat bubbles, expanding sections |
| Accent (10%) | Streamlit Primary Theme | Submit buttons, loading spinners, citations/links |
| Destructive | N/A | No destructive actions planned for Phase 4 |

Accent reserved for: the submit area button, inline citations, and `st.status` loaders.
